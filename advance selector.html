Advance Selector:

  • simple selector:

    1. tag selector
    2. class selector
    3. id selector
    4. universal selector

  • combinators selector:

    1. child selector (div p)
    2. direct child selector (div > p )
    3. adjustent sibling selector (div + p)
    4. general sibling selector (div ~ p)

  • attribute selector:

    1. tag[attr]
    2. tag[attr=val ]
    3. tag[attr^=val ]
    4. tag[attr|= val ]
    5. tag[attr$=val ]
    6. tag[attr*=val ]
    7. tag[attr~=val ]
